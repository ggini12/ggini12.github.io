프로젝트

중고책 거래 웹사이트 (JSP + MariaDB)
JSP와 MariaDB를 연동해 구현한 중고책 거래 플랫폼입니다.
도서 등록, 카테고리 및 가격대별 탐색, 바로 구매/장바구니, 회원가입 및 로그인, 마이페이지, 게시판(Q&A, 자유) 기능을 포함하여 실제 서비스 수준으로 개발했습니다.
웹 개발과 데이터베이스 연동 경험을 쌓은 대표 프로젝트입니다.
안드로이드 스튜디오 간단 앱들
Android Studio를 활용해 제작한 다양한 간단 앱들입니다.
설문조사 앱, To-Do 리스트, 여행 팁 앱 등 UI/UX 중심의 앱을 구현했으며, SQLite 및 그래프 표시 기능 등을 포함합니다.
모바일 앱 개발 역량과 사용자 경험 설계 경험을 쌓았습니다.
아두이노 기반 다국가 기상 정보 알림기
아두이노와 NodeMCU를 활용해 외부 기상 API와 연동하여 여러 나라의 날씨 정보를 실시간으로 수집하고 LED 디스플레이로 알림을 제공하는 IoT 프로젝트입니다.
하드웨어와 소프트웨어, API 연동을 융합한 프로젝트로, 실생활 문제 해결에 초점을 맞추었습니다.
