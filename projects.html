중고책 거래 웹사이트

JSP와 MariaDB를 활용한 중고책 거래 플랫폼 개발. 도서 등록, 카테고리 및 가격별 탐색, 장바구니, 바로구매 기능 구현.

  • JSP, Servlet 기반 웹 개발
  • MariaDB 데이터베이스 설계 및 연동
  • 회원가입/로그인 및 마이페이지 기능
  • 게시판(Q&A, 자유게시판) 구현

안드로이드 설문조사 및 To-Do 앱

안드로이드 스튜디오를 활용한 설문조사 앱과 To-Do 리스트 앱 개발. 사용자 편의성을 고려한 UI 구성 및 SQLite 활용.

  • 10개 이상의 질문으로 구성된 설문조사 앱
  • 사용자 답변 저장 및 그래프 결과 표시
  • To-Do 리스트 추가/삭제/수정 기능 구현

아두이노 기상 알림기

NodeMCU와 LED 모듈을 이용한 실시간 기상 알림 장치 개발. 외부 API 연동으로 다양한 나라의 날씨 정보 제공.

  • NodeMCU IoT 프로그래밍
  • OpenWeatherMap API 연동
  • LED를 통한 직관적 날씨 정보 표시

얼굴 인식 기반 드론 탐색 시스템

사용자가 업로드한 이미지와 실제 사람을 비교해 인식한 후, 해당 인물의 위치와 장소를 파악하여 알려주는 드론 프로젝트.

  • Tello 드론과 Python 기반 제어
  • 얼굴 인식(OpenCV) 및 실시간 카메라 영상 분석
  • 좌표 기반 위치 탐색 및 결과 출력
